## üåü Overview
<img align="right" width="450" src="https://user-images.githubusercontent.com/your-username/LayoodPad/raw/main/screenshot.png" alt="LayoodPad Screenshot" style="margin-left: 20px; border-radius: 10px; box-shadow: 0 4px 8px rgba(0,0,0,0.15);">

**LayoodPad** is a modern, intuitive online notepad application that makes text sharing and collaborative editing simple and secure. Just type a URL and start writing!

Created with simplicity in mind, LayoodPad offers:

- **Instant text sharing** - no account needed
- **Real-time syncing** across devices
- **Password protection** for sensitive notes
- **Code editor mode** with syntax highlighting
- **Unlimited notes** with no word limit
- **Clean, responsive interface** that works on any device

Whether you're sharing code snippets, writing notes, or collaborating on documents, LayoodPad provides a seamless experience with zero friction.

## üîß Installation

### Prerequisites

- Web server with PHP support
- MySQL database
- Composer (optional, for dependencies)

### Basic Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/DevDivyanshJindal/LayoodPad.git
   cd LayoodPad
   ```

2. **Configure your web server**
   - Point your web server to the project directory
   - Ensure PHP is enabled

3. **Database Configuration**
   - Create a MySQL database
   - Import the database structure from `database/schema.sql`
   - Configure database connection in `config.php`

4. **Start using LayoodPad**
   - Navigate to your local server URL
   - Start creating notes!

### Docker Setup (Alternative)

```bash
# Build the Docker image
docker build -t layoodpad .

# Run the container
docker run -p 8080:80 layoodpad

# Access LayoodPad at http://localhost:8080
```

## üìù Usage

### Creating a New Note

1. Go to the LayoodPad homepage
2. Enter a custom URL segment (e.g., `mynote`)
3. Click "Go" to create or access your note
4. Start typing - your note is automatically saved!

### Using Code Mode

1. Toggle the "Code Mode" switch
2. Start typing code
3. Line numbers will appear automatically
4. Use the copy button to copy the entire code

### Password Protection

1. Check the "Password Protect" checkbox
2. Enter a password in the input field
3. Click "Set" to protect your note
4. Share the URL - recipients will need the password to access

### Removing Password Protection

1. Access your protected note (enter the password)
2. Click "Remove Protection"
3. Your note is now publicly accessible
